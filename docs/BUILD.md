# How to Run

```
open src/index.html
```

That's it. No build step, no npm install, no server required. Works in any modern browser (Chrome, Firefox, Safari, Edge).

## Stack
- HTML5 / CSS3 / Vanilla JS
- [Dexie.js 3.2.7](https://dexie.org) — IndexedDB wrapper (bundled at `src/dexie.min.js`)

## Dev tips
- **DevTools → Application → IndexedDB → CTML** to inspect persisted data
- **DevTools → Application → Storage → Clear site data** to reset all data (simulates fresh install)
- Ctrl+S on the Capture screen saves the current idea
