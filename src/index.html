Szia!

Ez a CTML (Claude to My Life) — egy kísérleti személyes produktivitás app, amit épp tesztelek.

Az app négy napi rituáléra épül: Reggeli indítás → Ötlet rögzítés → Munka rajta → Esti lezárás. Minden adat a te készülékeden marad — nincs szerver, nincs fiók, nincs adatgyűjtés.

Ez egy alfa teszt — ha bármit furcsát találsz, vagy van ötleted, szólj bátran!

Telepítés

  • Android (Chrome): Menü (⋮) → „Hozzáadás a kezdőképernyőhöz" vagy „Alkalmazás telepítése"
  • iPhone (Safari): Megosztás gomb (□↑) → „Hozzáadás a Főképernyőhöz"
  • PC (Chrome/Edge): Kattints a címsorban lévő telepítés ikonra, vagy Menü → „Alkalmazás telepítése"

Telepítés után az app önálló alkalmazásként nyílik meg, és offline is működik.

Hey there!

This is CTML (Claude to My Life) — an experimental personal productivity app I'm currently testing.

The app is built around four daily rituals: Morning Boot → Capture → Work on It → Evening Wrap. All data stays on your device — no server, no account, no data collection.

This is an alpha test — if you find anything odd or have ideas, let me know!

How to install

  • Android (Chrome): Menu (⋮) → "Add to Home screen" or "Install app"
  • iPhone (Safari): Share button (□↑) → "Add to Home Screen"
  • PC (Chrome/Edge): Click the install icon in the address bar, or Menu → "Install app"

After installing, the app opens as a standalone app and works offline.
